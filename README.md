# Rubiks Cube Group and Markov Chains
This project investigates a group of permutations that represent the movements of a Rubiks Cube. We construct a Markov Chain with this group being the state space. Using the theory of Markov Chains, we show that it is impossible to fully scramble the cube i.e. it is impossible to rotate the cube so that all cube configurations are equally likely to be obtained in the long run. 

Please follow the instructions below. 
- Download the project folder from Git.
- Open Command Prompt/Terminal and change directory to the project folder.
- Create and activate a new virtual environment with the latest version of Python.
- Run "pip3 install -r requirements.txt" to install project requirements.
- Run "jupyter notebook" and click on the "RubiksCubeGroupDemo.ipynb" notebook.
- Run all cells.
